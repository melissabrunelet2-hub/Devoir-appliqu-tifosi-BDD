# Projet de base de données – Tifosi

## Objectif
Ce projet consiste à concevoir et manipuler une base de données MySQL pour un restaurant de type italien nommé *Tifosi*.

## Contenu du projet
- Création du schéma de la base de données
- Insertion des données
- Requêtes de vérification

## Structure du dépôt
- `data/` : fichiers de données utilisés pour l’insertion
- `sql/` :
  - `01_creation_schema.sql` : création des tables
  - `02_peuplement_donnees.sql` : insertion des données
  - `03_requetes_verification.sql` : requêtes de contrôle
- `make_inserts.py` : script Python utilisé pour générer les requêtes d’insertion
- `README.md` : description du projet

## Base de données
- Nom : `tifosi`
- SGBD : MySQL 8.0

## Auteur
Projet réalisé dans le cadre d’un devoir appliqué.
